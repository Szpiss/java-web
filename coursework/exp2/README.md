# 课内实验 2

本组整理的是课内实验中的 JSP + Bean 长方形计算实验。

## 实验要求

- 编写 `inputRectangle.jsp` 页面输入长方形的长和宽
- 将表单数据提交到 `showRectangle.jsp`
- 使用 JavaBean `Rectangle.java` 计算面积和周长
- 在结果页面显示长、宽、面积和周长

## 主要文件

- `src/main/java/org/example/bean/Rectangle.java`
- `src/main/webapp/rectangle/inputRectangle.jsp`
- `src/main/webapp/rectangle/showRectangle.jsp`
