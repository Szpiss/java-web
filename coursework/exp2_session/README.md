# 课内实验 2

本组整理的是课内实验 2，对应“内置对象 session 的应用”。

## 实验内容

- 使用 `session` 对象模拟在线考试系统
- 编写 `exam.jsp`、`exam_ans.jsp`、`exam_result.jsp`
- 在 `exam.jsp` 页面中答题
- 在 `exam_ans.jsp` 页面中显示答题结果
- 在 `exam_result.jsp` 页面中计算并公布考试成绩
- 额外编写 `visit_count.jsp`，利用 `application` 对象实现网站访问计数

## 主要文件

- `src/main/webapp/coursework/exp2_session/exam.jsp`
- `src/main/webapp/coursework/exp2_session/exam_ans.jsp`
- `src/main/webapp/coursework/exp2_session/exam_result.jsp`
- `src/main/webapp/coursework/exp2_session/visit_count.jsp`
- `coursework/exp2_session/report/javaweb开发基础课内实验二 2315302125 崔子霖.docx`
- `coursework/exp2_session/report/README.md`

## 运行结果

- `exam.jsp`：在线考试试卷页面
- `exam_ans.jsp`：答题结果展示页面
- `exam_result.jsp`：成绩计算与公布页面
- `visit_count.jsp`：网站访问计数页面
