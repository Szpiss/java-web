# 实验课 1

本组整理的是实验课第一次实验，对应《Java Web开发基础》实验指导书中的“实验 JSP应用”。

## 实验要求

- 在 `login.jsp` 输入姓名和年龄
- 提交到 `manger_control.jsp`
- 年龄大于等于 18 岁时转发到 `welcome.jsp`
- 年龄小于 18 岁时转发到 `forbid.jsp`
- `welcome.jsp` 和 `forbid.jsp` 配置错误处理页 `errorhandle.jsp`
- `login.jsp`、`welcome.jsp`、`forbid.jsp` 底部包含共同的友情链接内容

## 主要文件

- `src/main/webapp/lab/exp1/login.jsp`
- `src/main/webapp/lab/exp1/manger_control.jsp`
- `src/main/webapp/lab/exp1/welcome.jsp`
- `src/main/webapp/lab/exp1/forbid.jsp`
- `src/main/webapp/lab/exp1/errorhandle.jsp`
- `src/main/webapp/lab/exp1/links.jsp`
