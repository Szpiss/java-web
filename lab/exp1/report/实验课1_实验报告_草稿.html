实验名称实验课第一次实验：JSP 应用

一、实验目的

1. 掌握 JSP 基本语法。

2. 掌握在 JSP 页面中使用 include 标记动态加载文件的方法。

3. 学会使用 forward 动作实现页面跳转。

二、实验内容

本次实验根据实验指导书要求，完成了一个基于 JSP 的年龄判断应用。用户在 login.jsp 页面输入姓名和年龄后提交到 manger_control.jsp。该页面根据年龄判断用户是否已满 18 岁：若年龄大于等于 18 岁，则通过 forward 动作跳转到 welcome.jsp；若年龄小于 18 岁，则跳转到 forbid.jsp。另外，welcome.jsp 和 forbid.jsp 都配置了错误处理页面 errorhandle.jsp，并且 login.jsp、welcome.jsp、forbid.jsp 底部均通过 include 动态加载了共同的友情链接内容。

本次实验的主要页面如下：

1. login.jsp：输入姓名和年龄。

2. manger_control.jsp：根据年龄进行判断，并使用 <jsp:forward> 进行页面跳转。

3. welcome.jsp：显示成年用户欢迎信息。

4. forbid.jsp：显示未成年用户禁止访问的信息。

5. errorhandle.jsp：作为错误处理页面，显示异常信息。

6. links.jsp：存放友情链接，被多个页面动态包含。

核心代码如下：

<%@ page contentType="text/html; charset=UTF-8" language="java" errorPage="errorhandle.jsp" %>
<%
    request.setCharacterEncoding("UTF-8");

    String username = request.getParameter("username");
    int age = Integer.parseInt(request.getParameter("age"));

    if (age >= 18) {
%>
<jsp:forward page="welcome.jsp">
    <jsp:param name="username" value="<%= username %>" />
    <jsp:param name="age" value="<%= String.valueOf(age) %>" />
</jsp:forward>
<%
    } else {
%>
<jsp:forward page="forbid.jsp">
    <jsp:param name="username" value="<%= username %>" />
    <jsp:param name="age" value="<%= String.valueOf(age) %>" />
</jsp:forward>
<%
    }
%>

四、实验总结

通过本次实验，我进一步掌握了 JSP 页面开发的基本方法，理解了表单数据提交、页面接收参数、条件判断以及页面跳转的实现方式。在实验过程中，我重点练习了 <jsp:forward> 和 <jsp:include> 的使用方法，理解了 forward 用于服务器端页面跳转，而 include 可以实现公共页面片段的复用。

在编写页面时，我还学习了如何配置错误处理页面，使异常出现时能够统一跳转到 errorhandle.jsp，从而提高页面的健壮性。通过本次实验，我对 JSP 基本语法、页面指令和动作标签有了更清晰的认识，也为后续 Java Web 开发实验打下了基础。
